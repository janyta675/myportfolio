Python Based Projects

Recepe Search Engine(Python-Django Framework)

I developed a recipe search engine using Python Django framework and a recipe API. The application allows users to search for recipes based on various criteria, such as ingredients, cuisine, and dietary restrictions. Users can also save their favorite recipes, rate and review recipes, and create their own recipe collections. The search engine integrates with an external recipe API, which provides a vast database of recipes and ingredients. The API returns recipe data in JSON format, which the application parses and displays to the user in a user-friendly format.


Monthly Challenges(Python-Dictionary)

For my first Python project, I designed and developed a website that showcases monthly fitness challenges. The website provides users with a fun and engaging way to improve their fitness levels by challenging themselves to complete specific exercises and routines. I used HTML, CSS, and JavaScript to create the frontend, while Python handled the backend operations. I decided to use a dictionary as a simple and easy-to-maintain alternative to a traditional database management system, as I wanted to focus on learning Python and web development concepts.

Gym Website(Python-Django Framework)

The website features a modern and user-friendly design that reflects the gym's brand and values. I used HTML, CSS, and JavaScript to implement the frontend, while Python Django framework handled the backend operations. The website also includes a blog section that provides valuable information about nutrition, exercise, and healthy lifestyle habits. I wrote several blog posts myself, but also provided the gym staff with a user-friendly interface to create and publish their own content.
